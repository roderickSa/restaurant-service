# Restaurant API

### General info

<strong>Technologies</strong>:

<ol>
  <li>nestjs</li>
  <li>postgres</li>
</ol>

### 1. Clonar proyecto

### 2. `yarn install`

### 3. Clonar el archivo `.env.template` y renombrarlo a `.env`

## 4 Levantar la base de datos

```
docker-compose up -d
```

### 5 Levantar: `yarn start:dev`

```
http://localhost:3000/graphql
```
